# Fashion Recommendation System

## Overview

This project builds an image-based fashion recommendation system using deep learning and computer vision techniques. The system recommends visually similar fashion products based on an uploaded image.

## Data Source

https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-small

## Workflow

1. Image preprocessing
2. Feature extraction using ResNet50
3. Feature normalization
4. Cosine similarity calculation
5. Recommendation generation

## Key Features

* Deep learning-based image analysis
* Content-based recommendation system
* Visual similarity search
* Interactive user interface

## Results

The system successfully identifies and recommends visually similar fashion items by comparing deep image features extracted from a pre-trained ResNet50 model.
